# Chrome Dino Controller
### An Arduino Controller for the Google Chrome Dinosaur Game

Arduino components needed are:
```
1 - Arduino Leonardo
1 - Ultrassonic Sensor (HC-SR04)
1 - Buzzer (Optional)
1 - Dinosaur
```
